# Bibliothèque de métaphores visuelles

Ce fichier catalogue les **objets métaphoriques** à utiliser dans les headers, organisés par thème d'article. Consulter au moment de choisir les 3-4 objets qui composeront la zone droite du header.

## Principes généraux

1. **3 à 4 objets par image** — pas plus, pas moins
2. **Objets complémentaires** qui racontent une mini-histoire (pas 4 objets aléatoires)
3. **1 objet dominant** (220-280px) + **2 secondaires** (120-160px) + optionnellement **1 discret** (80-100px)
4. **Tilts variés** (5-15°) pour éviter la rigidité
5. **Cohérence de style** : tous flat 2D pur, tous avec le même niveau de détail
6. **Couleurs réparties** selon la palette : 1-2 objets en cobalt, 1 en violet, 1 en teal ou navy

**Règle d'or** : une **bonne métaphore** vaut 4 objets décoratifs. Si tu hésites, simplifier (3 objets forts > 4 objets moyens).

## Bibliothèque par thème

### 🧠 Intelligence artificielle / LLM / Machine Learning

**Objets pertinents** :
- **Livre ouvert** avec texte stylisé (connaissance, apprentissage)
- **Cerveau stylisé** (neurones, mais flat et géométrique, **pas** cartoon)
- **Lignes/nœuds connectés** (réseau neuronal abstrait)
- **Prisme** (transformation d'information)
- **Œil stylisé** géométrique (vision, perception)
- **Cristal/gemme géométrique** (structure de données)

**Combinaisons qui fonctionnent** :
- *Livre + loupe + document + petit graphique* → lecture/compréhension (idéal RAG, recherche)
- *Prisme + rayons + points dispersés* → transformation, génération
- *Cerveau + réseau de nœuds + data* → architecture ML

## Comment choisir pour un article spécifique

**Étape 1** — Identifier le **thème principal** (peut croiser plusieurs catégories de la bibliothèque).

**Étape 2** — Choisir **1 objet dominant** qui incarne le cœur du sujet.

**Étape 3** — Ajouter **1-2 objets de support** qui précisent ou enrichissent.

**Étape 4** — Optionnellement **1 objet discret** pour équilibre visuel (souvent data-viz simplifiée ou cluster de points).

### Exemple de choix

*Article : "RAG pour les PME : 5 mythes à oublier"*

- **Thème principal** : IA + recherche documentaire (RAG = Retrieval Augmented Generation)
- **Thèmes croisés** : IA/LLM + Recherche/SEO
- **Choix** :
  - Objet dominant : **livre ouvert** (connaissance indexée)
  - Secondaire : **loupe** (recherche/retrieval)
  - Secondaire : **document plié** (contenu récupéré)
  - Discret : **3 petits points teal** (dispersion des mythes)
  - Focal : petit badge orange avec **"5"** (les 5 mythes)

### Exemple 2

*Article : "Comment un chatbot peut gérer 80% de vos demandes"*

- **Thème principal** : Chatbots + Automatisation
- **Choix** :
  - Objet dominant : **bulle de dialogue large** (conversation)
  - Secondaire : **horloge 24h** (disponibilité permanente)
  - Secondaire : **engrenages** (automatisation)
  - Focal : petit badge orange avec **"80%"** (ou **"24/7"**)

### Exemple 3

*Article : "Rédiger 50 courriels de prospection avec ChatGPT"*

- **Thème principal** : Vente + IA
- **Choix** :
  - Objet dominant : **enveloppe** (courriel)
  - Secondaire : **plume / stylo** (rédaction)
  - Secondaire : **entonnoir** (prospection)
  - Focal : petit badge orange avec **"50"** ou **"×10"** (volume)

## Métaphores à éviter absolument

- ❌ **Ampoule** pour "idée" (cliché absolu)
- ❌ **Rouages qui s'emboîtent** pour "mécanisme" (cliché)
- ❌ **Poignée de main** très figurative (sauf si extrêmement stylisée)
- ❌ **Robot/mascotte** pour "IA" (tombe dans le cartoon)
- ❌ **Cerveau avec circuits imprimés** (cliché "AI 2015")
- ❌ **Graphique qui monte en flèche géante** (caricatural)

**Règle** : si l'objet est dans les 10 premières images Google pour "IA illustration", c'est probablement un cliché. Chercher plus spécifique et concret.
